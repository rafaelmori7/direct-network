# Direct Network — Site

## Arquivos
- `index.html` — Home com grid de eventos e filtros
- `evento.html` — Página individual de evento (template)
- `logo_direct.png` — Logo da marca

## Como usar no Framer

### Opção A — Colar como componente de código
1. No Framer, crie um novo projeto
2. Vá em "Assets" > "Code" > "New Code File"
3. Cole o conteúdo de cada arquivo como componente

### Opção B — Usar como referência visual
Use os arquivos HTML como referência fiel de design
para recriar no Framer usando os componentes nativos + CMS.

## CMS — Campos por evento
Crie uma Collection chamada "Eventos" com os seguintes campos:
- Nome (text)
- Data (date)
- Cidade (text)
- Local/Venue (text)
- Gênero (option: Eletrônica / Pagode / Sertanejo / Show)
- Descrição (rich text)
- Link afiliado (link)
- Flyer (image)
- Plataforma (option: Sympla / Ingresse / Outros)

## Cores da marca
